Alkali

Overview
Alkali is a multi-diciplinary arts channel with an aim to not bias certain formats of art. This is a branding project where we are asked to create a visual identity based on attributes set by our lecturers. I wanted to explore how to keep the playing field level within the various formats of art and decided upon creating a wireframe/mesh look with greyscale colours letting the content of the channel include the art, and the visual identity include the base platforms used when creating structures. I treated this project as quite an experimental outlet allowing me to explore things like 3d and linking audio and visuals.
Brief
We were asked to create a visual identity based around 3 brand values and a target audience. I decided to chose ‘Arts Channel aimed at all genders 25—45, Fluid Dangerous, and Conceptual’. We were asked to create a 30 second ident with 15 second living hold alongside a visual identity created for the channel, and any supporting media like ‘stings’, ‘intersituals’ or, ‘on—screen menus’.
Process
I started this project by exploring the brand values and how I could incorporate them within an arts channel. I looked into the periodic table which gave me the inspiration for the logo and channel name. Alkali being dangerous (causes chemical burns at high concentrations) and usually dissolved in fluid. The wireframe look came from stripping a material down to it’s most basic form, which would be it’s atomic. structure. Each vertex the edge would be linked to, are it’s atomic structure. Building upon this would add surface properties like colour, reflectance or texture, etc. For the visual identities font I chose a mono-spaced font for body text. I went with this as it’s typically used in non-graphical (command line) user interfaces which for similar reasons to the wireframe, I wanted to visualise the most base forms of media.
